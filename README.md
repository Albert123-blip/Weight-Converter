# ⚖️ Java Weight Conversion Program

A simple Java console application that converts weight between pounds (lbs) and kilograms (kgs). This project was created to practice Java fundamentals such as user input, conditional statements, variables, and mathematical calculations.

## Features

* Convert pounds (lbs) to kilograms (kgs)
* Convert kilograms (kgs) to pounds (lbs)
* Console-based user interface
* Displays an error message for invalid menu choices

## Technologies Used

* Java
* IntelliJ IDEA

## How It Works

1. The program displays a menu with two conversion options.
2. The user selects a conversion by entering `1` or `2`.
3. The user enters the weight to convert.
4. The program performs the calculation and displays the converted weight.
5. If an invalid option is entered, the program prompts the user to enter a valid choice.

## Example

```text
Weight Conversion program
1: Convert lbs to kgs
2: Convert kgs to lbs

Choose an option: 1
Enter the new weight in lbs: 150

The new Weight is: 68.04kgs
```
